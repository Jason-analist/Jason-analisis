# Análisis de Impacto: Ladrillos PET en Santander

**¿Cuántas casas se pueden construir con el plástico que botamos?**

### Problema

Santander genera 372 toneladas de PET al mes. El 90% termina en rellenos o ríos.

### Solución Propuesta

Convertir PET en eco-ladrillos para vivienda VIS. 1 ladrillo = 2kg PET.

### Stack

SQL, Python, Pandas, Matplotlib

### Hallazgos Clave

- Con el PET de 1 año en Bucaramanga: 29 casas VIS
- En todo Santander: **148 casas VIS anuales** = techo para 592 personas
- Reducción CO2: 1,120 toneladas anuales vs ladrillo tradicional

### Cómo replicar

1. Clona repo
2. Corre `sql/consultas.sql` en PostgreSQL/MySQL
3. Ejecuta `notebooks/analisis.ipynb`
